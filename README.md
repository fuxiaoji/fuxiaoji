## 🚀 精选项目 (Featured Projects)

### 📈 量化与数据分析
- **[reprodue_fama_modle](https://github.com/fuxiaoji/reprodue_fama_modle)**: 完整复现 Fama-French 三因子模型的学术级项目，使用 2000-2026 年数据进行实证分析，包含完整数据处理、回归检验与可视化流程。
- **[quant_test](https://github.com/fuxiaoji/quant_test)**: 结合 DeepSeek LLM 的量化轮动策略回测系统，实现 ETF 动量轮动分析与可视化报告生成。
- **[mcm](https://github.com/fuxiaoji/mcm)**: 2026 年美国大学生数学建模竞赛 (MCM) C 题完整解题方案与数据分析代码。

### 🕸️ 爬虫与自动化
- **[tieba_spider](https://github.com/fuxiaoji/tieba_spider)**: 高效的百度贴吧全量数据抓取工具，支持楼层、楼中楼对骂记录及图片资源的并发下载。
- **[crawler_for_biddingprofile](https://github.com/fuxiaoji/crawler_for_biddingprofile)**: 重庆公共资源交易网中标结果增量抓取系统，自动分页解析并导出结构化数据。

### 💻 全栈与 Web 开发
- **[Home_Page](https://github.com/fuxiaoji/Home_Page)**: 基于 Flask 的现代化个人主页系统，集成 GitHub API 展示动态数据，支持多种部署方案。
